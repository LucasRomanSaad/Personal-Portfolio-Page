Project Overview

The "Tenzies" app is an engaging browser-based dice game built using React, offering players an interactive gaming experience where they roll dice to achieve the coveted "Tenzies" combination. Players strategically freeze dice by clicking on them, aiming to match all dice to the same value. The game provides visual feedback with celebratory confetti animations upon achieving "Tenzies" and highlights held dice for easy tracking. Additionally, players can track game statistics such as the number of rolls and elapsed time, adding a competitive edge to the gameplay. The app features a responsive design, ensuring seamless play across various devices.

Overall, the "Tenzies" game app offers users an enjoyable gaming experience, featuring interactive dice rolling and strategic gameplay. Built with React and CSS, it ensures a seamless and engaging experience across different devices.
